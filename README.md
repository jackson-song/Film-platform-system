# vue-douban

> A Vue.js project

## 技术栈

vue + vuex + vue-router + axios + webpack + ES6

运行前请先确认电脑上已经安装好node环境

 - 克隆项目到本地

```shell
git clone https://github.com/nh0007/vue-douban.git
```

 - 进入项目根目录，在命令行运行语句安装依赖：

```shell
npm install
```

 - 运行项目

```shell
npm run dev
```

 - 等待运行完毕后，在浏览器输入命令行提示地址，可看到项目效果。
